# What is this?

Official Apache httpd image modified to have SSL turned on. Use the built
in cert/key, copy in your own cert and key into image or mount them as volumes.

> Note: The included cert/key was generated by me for playing. Don't use this
> live!

# How to use this image

The default way:
```
docker run -d -p 443:443 inanimate/httpd-ssl
```

The more configurable way:

```
$ docker run -d -e SERVER_NAME=foobar.example.com \
-v /path/to/server.crt:/usr/local/apache2/conf/server.crt \
-v /path/to/server.key:/usr/local/apache2/conf/server.key \
inanimate/httpd-ssl
```
